== Description ==

New Kid is a modern WordPress block theme that embraces the NeoBrutalism design movement. With its bold borders, dramatic shadows, and vibrant color palette, this theme makes a statement while remaining fully customizable through WordPress's Full Site Editing capabilities.

**Key Features:**
* Full Site Editing (FSE) support with comprehensive block styling
* NeoBrutalism aesthetic with bold borders and dramatic drop shadows
* Customizable shadow controls - enable/disable shadows per block
* Dynamic shadow scaling based on border width
* Border radius support with matching shadow corners
* Custom block styles: Default, Highlight, Alert, and Success
* Responsive design optimized for all devices
* Custom color palette inspired by NeoBrutalism design principles
* Google Fonts integration (Stint Ultra Expanded, Lexend Mega, Proza Libre, Public Sans)
* Cache-busted assets for optimal performance
* Translation ready with .pot file included
* RTL language support
* Accessibility ready - WCAG 2.1 AA compliant
* Keyboard navigation support
* Screen reader optimized

**Perfect for:**
* Creative agencies and portfolios
* Bold brand websites
* Art and design blogs
* Modern business sites
* Anyone who wants to stand out from the crowd

The theme provides extensive customization options while maintaining its distinctive NeoBrutalism character. Users can adjust colors, spacing, borders, and shadows using WordPress's native block editor controls.

== Installation ==

1. Download the theme zip file
2. In your WordPress admin, go to Appearance > Themes
3. Click "Add New" then "Upload Theme"
4. Choose the zip file and click "Install Now"
5. Activate the theme
6. Go to Appearance > Editor (Beta) to start customizing with Full Site Editing

== Frequently Asked Questions ==

= Is this theme compatible with page builders? =
New Kid is designed for WordPress's native Full Site Editing experience. While it may work with page builders, it's optimized for the block editor and FSE.

= Can I disable the NeoBrutalism shadows? =
Yes! You can disable shadows on any block using the border settings panel in the block editor. Uncheck the shadow option to remove shadows while keeping the bold borders.

= Are the colors customizable? =
Absolutely! The theme includes a custom color palette, but you can change any colors using the block editor's color controls or add your own custom colors.

= Does this theme work with WooCommerce? =
The theme includes basic styling for WooCommerce blocks and should work well with e-commerce functionality.

== Customization ==

**Block Styles:**
* Default - Clean NeoBrutalism styling with borders and shadows
* Highlight - Pale yellow background for important content
* Alert - Orange background with bold typography for warnings
* Success - Green background for positive messages

**Color Palette:**
The theme includes a carefully curated color palette inspired by NeoBrutalism:
* Black and White (primary)
* Neo Orange, Neo Blue, Neo Green (brand colors)
* Sky Blue, Pale Red, Pale Violet, Soft Blue, Pale Cyan, Pale Yellow (accent colors)
* Bright Red, Bright Green, Bright Yellow, Bright Magenta, Bright Cyan (highlight colors)

**Typography:**
* Headings: Impact, Arial Black (bold and impactful)
* Body: Courier New (monospace for technical feel)
* Special fonts: Stint Ultra Expanded, Lexend Mega, Proza Libre, Public Sans
